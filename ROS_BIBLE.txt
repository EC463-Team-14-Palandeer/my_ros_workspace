GENERAL PACKAGE IDEA:

        Now we get into what you'll need to know for how everything launches

        Our launch file (my_robot_bringup.launch.py) determines which nodes we need to bring up. These nodes include the parameters...
                                                                                                  Package -> Package it is in
                                                                                                  Executable -> The file which holds logic for node
                                                                                                  Name -> What node will be called

        You shouldn't have to worry about name, but Package and Executable are muy importante

        THE PACKAGE name is defined by which package it sits in (my_robot_bringup, pico_comms, or robo_cayote_control)
        THE EXECUTABLE name is defined by what the file is referred to in the setup file. 
                Ex. robo_cayote_control/setup.py -> Go to bottom to entry_points ->  'mqtt_ack_node = robo_cayote_control.mqtt_ack_node:main',    <--- mqtt_ack_node is the name for the executable
                                                                                     'ris_go2rtc_node = robo_cayote_control.ris_go2rtc_node:main',
                                                                                     'cayote_rl_brain = robo_cayote_control.cayote_rl_brain:main',
                                                                                     'yolo_processor = robo_cayote_control.yolo_processor:main',
                                                                                     'mission_controller = robo_cayote_control.mission_controller:main'

        And then we add a lil pizzazz to each Node and we're good :D. These nodes are already made and shouldn't need to be edited (at least too much). But, what do I know? I'm a loser chud...

        This "pizzazz" can be added either in the Node itself, or be defined in a .yaml file. If it's in the yaml file, you just have to add that to the node
                                                                                                  Ex. Node(package='nav2_controller', executable='controller_server', name='controller_server',
                                                                                                  parameters=[nav2_config, {'use_sim_time': use_sim_time}]),
                                                                                                  ^
